# Restaurant Analysis Dashboard

## Overview

This Tableau dashboard analyzes restaurant data using interactive KPIs and visualizations to provide insights into restaurant ratings, popularity, pricing, and customer preferences.

## Tools Used

* Tableau
* Excel
## Dashboard Interaction : <a href = "https://github.com/Rameshvari-Kedari/Restaurant_analysis_Tableu/blob/main/Restaurant%20Analysis.twbx">View Dashboard</a> 
## Dashboard Features

* Restaurant Ratings Analysis
* Popularity Analysis
* Price Range Analysis
* Location Analysis
* Customer Votes Analysis
* Interactive Filters and KPIs

## Dataset

Restaurant Analysis Dataset

## Key Insights

* Identifies the highest-rated restaurants.
* Compares restaurants across different price ranges.
* Analyzes the popularity of various cuisines.
* Shows restaurant distribution by location.
* Highlights customer preferences based on ratings and votes.
