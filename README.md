# Restaurant Analysis Dashboard

## Overview

This Tableau dashboard analyzes restaurant data using interactive KPIs and visualizations to provide insights into restaurant ratings, popularity, pricing, and customer preferences.

## Tools Used

* Tableau
* Excel

## Dashboard Features

* Restaurant Ratings Analysis
* Popularity Analysis
* Price Range Analysis
* Location Analysis
* Customer Votes Analysis
* Interactive Filters and KPIs

* Identifies the highest-rated restaurants.
